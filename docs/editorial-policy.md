# Editorial & Update Governance Policy

## Principle
Every update answers: Is it true? Is it academically relevant? Is it appropriately sourced? Is it safe to publish? Does it strengthen a coherent academic identity? If any answer is uncertain, verify before publishing.

## Content classification
- **Canonical facts** (name, ORCID, degrees, completed publications/projects/supervision, patents, historical positions, awards): stored as stable data after verification.
- **Time-sensitive facts** (current role, group membership, ongoing grants, current service, citation metrics): carry a verification date and a review rule; rechecked on the monitor schedule.
- **News candidates** (accepted papers, talks, panels, mentions): editorial screening; usually enter via pull request.
- **Excluded**: family, private travel, home address, private phone, health, personal finance, political/religious identity, unrelated social content, confidential review/grant/student data.

## Confidence-based governance
- **High confidence → may auto-merge after all tests pass**: DOI/Anthology metadata, official publication links, corrected venue/pages, stable ORCID records, official GitHub release links, broken-link replacement from an authoritative source, sitemap/feed regeneration.
- **Medium confidence → pull request for review**: new paper via secondary source, invited talk, award, workshop participation, group/institutional news, social-media item, collaborator announcement, new student/researcher info, role change.
- **High impact → explicit human approval**: biography rewrites, funding amounts, role titles, major impact claims, contact changes, photographs, supervision-record changes, leadership claims, student employment outcomes, press quotations, removal of historical records, substantial design changes.

## Editorial standards
EU English; accurate academic terminology; modest, evidence-based claims; consistent perspective; clear dates and complete names; correct institutional titles. Avoid "groundbreaking / revolutionary / world-leading / game-changing / unprecedented" unless an authoritative source justifies it. Prefer factual formulations ("The paper was accepted at…", "Dr Afli contributed to…"). Never invent references, collaborators, awards, funding, titles, projects, publication status, or impact.

## Citation metrics
Shown only with named source and retrieval date; never presented as permanent; never merged across databases; never used as the dominant measure of quality.
